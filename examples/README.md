# Skill Showcases (Before/After)

These examples show the expected impact when a skill is applied well.

## 1. repo-explainer
- Before: New contributors need hours to understand module boundaries.
- After: AI agent outputs architecture map and onboarding path in minutes.

## 2. unit-test-generator
- Before: Critical paths have low coverage and brittle manual tests.
- After: Deterministic tests added for core services with clear fixtures.

## 3. flaky-test-detector
- Before: CI fails intermittently with no root-cause trail.
- After: Flaky signatures identified with quarantine + stabilization plan.

## 4. github-actions-generator
- Before: Inconsistent local-only checks and ad-hoc releases.
- After: Standard lint/test/build/release workflows with branch protection fit.

## 5. dockerfile-generator
- Before: Large images, slow builds, and inconsistent runtime behavior.
- After: Multi-stage builds, smaller images, reproducible production runtime.

## 6. rag-pipeline-builder
- Before: Retrieval quality unclear and hallucinations unmeasured.
- After: End-to-end RAG flow with chunking strategy, eval set, and guardrails.

## 7. llm-cost-optimizer
- Before: Token spend rises without visibility or routing policy.
- After: Prompt compression, caching, and model routing reduce cost per request.

## 8. dependency-vulnerability-scanner
- Before: Vulnerability checks are manual and sporadic.
- After: Repeatable scans integrated into CI with prioritized fix output.

## 9. accessibility-auditor
- Before: UI ships with keyboard and contrast regressions.
- After: Actionable a11y findings plus remediations tied to WCAG issues.

## 10. api-endpoint-generator
- Before: API implementation and docs drift over time.
- After: Endpoint, schema, and OpenAPI artifacts generated together.
