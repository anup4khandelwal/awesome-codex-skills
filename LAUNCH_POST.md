# Launch Post Draft: 100 Codex Skills for AI Engineers

Today I open-sourced 100-codex-skills-for-ai-engineers.

A production-ready library of reusable Codex Skills for AI coding agents working across:
- Node.js
- TypeScript
- Python
- LangGraph
- RAG systems
- DevOps and GitHub Actions
- Security, data, frontend, and backend workflows

## Why this exists

Most agent prompts are one-off and inconsistent.
This repo provides structured skill modules with explicit inputs, outputs, workflow steps, guardrails, and limitations.

That means better reliability, less prompt drift, and faster delivery.

## What is inside

- 100 skills across 9 categories
- Standard skill format for easy contribution
- CI checks for structure and docs quality
- Showcases and good-first-issue drafts for contributors

## Try it

```text
/skill repo-explainer
```

```text
/skill rag-pipeline-builder
```
