# Contributing Guide

## Adding a Skill
1. Create `skills/<category>/<skill-name>/`.
2. Add `SKILL.md`, `README.md`, `examples.md`, and `scripts/`.
3. Keep the required SKILL.md section structure.
4. Add practical prompt examples and guardrails.
5. Submit a PR with validation notes.

## Quality Bar
- Scope is clear and automation-focused.
- Inputs/outputs are machine-actionable.
- Steps are deterministic where possible.
- Guardrails and limitations are explicit.
