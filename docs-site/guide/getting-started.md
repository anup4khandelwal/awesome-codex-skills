# Getting Started

1. Choose a skill directory under `skills/<category>/<skill-name>/`.
2. Copy a prompt from that skill's `examples.md`.
3. Run in your agent and validate outputs with tests and CI.

## Recommended First Skills

- `repo-explainer`
- `unit-test-generator`
- `github-actions-generator`
- `rag-pipeline-builder`
- `dependency-vulnerability-scanner`
